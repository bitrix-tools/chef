# Начало работы

## Установка

```bash
npm install -g @bitrix/chef
```

## Быстрый старт

Инициализируйте окружение для сборки:

```bash
chef init build
```

Создайте и соберите первое расширение:

```bash
chef create my.extension
chef build my.extension
```

## Требования

- Node.js >= 22
- Проект на Bitrix или директория с исходниками модулей
